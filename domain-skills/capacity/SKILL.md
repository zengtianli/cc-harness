# 水资源承载力评价项目知识

> 路径：`.claude/skills/capacity/`

## 业务概述

绍兴市水资源承载力评价，采用组合赋权 TOPSIS 模型。包含年度评价和季度动态评价两套体系。

## 评价体系

### 指标体系（14项，年度）

| 准则层 | 指标 | 说明 |
|--------|------|------|
| 水资源禀赋 | C1, C2 | 资源总量与人均 |
| 水资源保障 | C3, C4 | 供水保障能力 |
| 水资源节约 | C5-C9 | 用水效率（万元GDP、工业、农田灌溉等） |
| 水资源保护 | C10-C12 | 水功能区达标、污水处理等 |
| 水资源管理 | C13, C14 | 管理制度执行 |

### 季度评价指标（8项）

C1-C8，用于实时监测与动态评价。

## 计算方法

### 权重确定（组合赋权）

1. **AHP**（层次分析法）：主观权重，专家打分
2. **CRITIC**：客观权重，基于标准差和相关系数
3. **组合权重**：`W = 0.5 × W_AHP + 0.5 × W_CRITIC`

### 评价模型

- **年度**：极差标准化 → 加权求和 → WREI（水资源综合评价指数）
- **季度**：向量标准化 → TOPSIS → 相对接近度排名

## 运行方式

```bash

cd sxczl/quarter/v1 && python main.py ./

cd sxczl/year/v1 && python main.py ./
```

## 数据文件格式

- 制表符分隔的 TXT 文件
- 第一列：指标编号（C1-Cn）
- 第二列：性质（1=正向，-1=负向）
- 后续列：各区域数值
