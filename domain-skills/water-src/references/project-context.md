# 水源地自评估项目上下文

## 项目概述

这是**2025年仙居县水源地安全保障达标自评估**项目，涉及盂溪水库和西岙水库的年度评估材料整理。

## 核心工作

1. **材料更新**：将2024年材料更新为2025年版本
2. **年份替换**：2024->2025, 2023->2024
3. **盖章流程**：协调水利局、环保局、水库管理公司
4. **质量检查**：印章检测、文件对比

## 评估对象

| 水库 | 管理公司 | 评分目标 |
|-----|---------|---------|
| 盂溪水库 | 仙居县盂溪水库发展有限公司 | 100分 |
| 西岙水库 | 浙江永安水务集团有限公司 | 100分 |

## 文件命名规范

```
{指标编号} {年份}_{水库名}_{说明}.{扩展名}
```

示例：
- `1-1 2025_盂溪_来水量保证率计算说明.docx`
- `5-1 2025_西岙_取水口水质达标率计算说明.pdf`

## 目录结构

| 目录 | 用途 |
|-----|------|
| `00 源文件/` | 原始 Word 文档 |
| `盂溪水库/` | 盂溪水库整理后 PDF |
| `西岙水库/` | 西岙水库整理后 PDF |

## 现有脚本

| 脚本 | 功能 |
|-----|------|
| `detect_stamps.py` | 检测 PDF 中的印章 |
| `export_pdfs.py` | Word 导出为 PDF |
| `organize_docx.py` | 按指标整理文档 |
| `generate_compare.py` | 生成文件对比清单 |
| `compress_pdfs.py` | 压缩 PDF 文件 |

## Python 编写规范

1. **备份优先**：覆盖文件前必须备份，格式 `原文件名.YYYYMMDD_HHMMSS.扩展名`
2. **路径处理**：使用 `pathlib.Path` 处理中文路径
3. **编码**：统一使用 UTF-8
4. **日志**：操作过程输出到控制台和日志文件
