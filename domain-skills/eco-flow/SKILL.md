# 生态流量项目知识

> 路径：`.claude/skills/zdwp-eco-flow/`

## 业务概述

小型水库生态流量核定与保障实施方案编制。依据水利部（水资管〔2024〕313号）和浙江省水利厅（浙水资〔2025〕12号）要求，逐工程明确生态流量目标并落实保障措施。

## 标准工作流程

### 1. 工程排查（全貌→聚焦）

- 全面排查县域内**所有水库**（含大中型），列出完整清单
- 区分"已核定"（大中型水库核定、小水电核定中已完成）和"未核定"
- 筛选链必须完整展现：总数 → 已完成数 → 剩余数 → 排除数 → 最终核定数
- 排除原因必须引用政策依据（如"集雨面积不足0.5km²"）

### 2. 生态流量计算

- **主要方法**：Tennant 法（多年平均流量的10%）、Qp 法（90%保证率最枯月平均流量）
- **数据要求**：长序列水文资料（n≥30年），景宁项目用1962-2023年（62年）
- **径流推求**：缺实测资料时，用参证站建立降雨径流产汇流模型，面积比拟法推求
- **敏感期生态流量**：需调查下游是否有洄游鱼类、珍稀物种、自然保护区等，如无则说明"不涉及"
- **小流量处理**：当生态流量值 < 0.01 m³/s 时，采用年生态水量替代

### 3. 保障措施

- **泄放设施**：先介绍每座水库的工程现状（位置、管理单位、放水建筑物），再提改造方案
- **监测设施**：电磁流量计 + 动态视频，数据接入省水文通信平台
- **调度方案**：防洪安全为前提 → 优先生活用水 → 保障生态用水 → 统筹生产用水
- **运行机制**：明确核定、泄放、监测、调度、监督各环节的责任主体

## 写作注意事项（来自景宁项目经验）

1. **筛选链不能断**：从44座→已完成41座→剩余3座→排除1座→核定2座，每一步都要交代清楚
2. **先介绍后方案**：4.1节必须先写每座水库的现场查勘情况（位置、集雨面积、管理单位、现有设施），再提改造建议
3. **内容不留空白**：每个章节标题下必须有完整内容，不能只写标题或半句话
4. **乙方立场**：用"建议""按要求确定"，不用"必须""不再列入"等定论式表述
5. **委婉用词**："缺乏在线监测手段"而非"监管盲区"，"难以有效监测"而非"无法监测"
6. **数据要有分析**：时间序列数据要做趋势分析，不只列表
7. **质量检查**：写完后用 `report-writing` skill 的六维框架（TLAPDS）逐维度检查

## 关键政策文件

- 水利部（水资管〔2024〕313号）：全面开展河湖和已建水利水电工程生态流量确定与保障
- 浙江省（浙水资〔2025〕12号）：2025年底完成3000座以上，2026年6月底前完成省内全部
- 水利部细化要求（征求意见稿）：小（2）型水库、无取水许可、坝下无天然河道的可暂不纳入
