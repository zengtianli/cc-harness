# 共享资源 (~/Work/resources/)

水利公司级共享数据，多个项目复用。读取时用绝对路径 `~/Work/resources/...`。

## 数据 (data/)

| 路径 | 内容 |
|------|------|
| `data/全省水资源基础数据/原始数据/Excel/` | 全省水库、河流、水闸基础数据 xlsx |
| `data/全省水资源基础数据/原始数据/GeoJSON/` | 水利要素 GeoJSON |
| `data/全省水资源基础数据/原始数据/Shapefile/` | 水利要素 shp |
| `data/全省水资源基础数据/工程数据/` | 工程筛选和统计 |
| `data/原始年报/` | 2019-2024 全省水资源分区基本情况表 xlsx |

## GIS (gis/)

| 路径 | 内容 |
|------|------|
| `gis/水利要素/水库/` | 水库点位 shp |
| `gis/水利要素/` | 河流、水闸等水利要素 |
| `gis/行政边界乡镇/` | 乡镇级行政区划 shp（CGCS2000）|
| `gis/专题数据/洪水风险/` | 洪水风险专题 |
| `gis/专题数据/岸线规划/` | 岸线规划专题 |
| `gis/专题数据/生态流量/` | 生态流量专题 |
| `gis/专题数据/供水管网/` | 供水管网专题 |
| `gis/小水电/` | 小水电 geojson |
| `gis/其他.gdb/` | ArcGIS GDB（杂项空间数据）|

## 参考资料 (references/)

| 路径 | 内容 |
|------|------|
| `references/规范政策/` | 行业规范、政策文件 |
| `references/规划报告/` | 规划报告 |
| `references/内部资料/` | 公司内部模板、字体、软著 |
| `references/数据资源/` | 数据资源索引 |

## 约束

- GIS 坐标系统一为 CGCS2000，输出可转 WGS84
- 大文件（shp/gdb/xlsx）不要 git track，只在本地使用
- 修改共享数据前确认不影响其他项目
